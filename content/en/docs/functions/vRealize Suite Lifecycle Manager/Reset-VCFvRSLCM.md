# Reset-VCFvRSLCM

### Synopsis
Redeploy vRealize Suite Lifecycle Manager

### Syntax
```
Reset-VCFvRSLCM
```

### Description
The Reset-VCFvRSLCM cmdlet redeploys the existing vRealize Suite Lifecycle Manager

### Examples
#### Example 1
```
Reset-VCFvRSLCM
```
This example redeploys the vRealize Suite Lifecycle Manager
