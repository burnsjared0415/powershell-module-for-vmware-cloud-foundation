---
title: "Get-VCFDepotCredential"
weight: 1
description: >
    Retrieves the configured My VMware credentials for the depot
---

## Syntax
``` powershell
Get-VCFDepotCredential
```

## Examples
### Example 1
``` powershell
Get-VCFDepotCredential
```
Retrieves the My VMware credentials that have been configured for the depot
