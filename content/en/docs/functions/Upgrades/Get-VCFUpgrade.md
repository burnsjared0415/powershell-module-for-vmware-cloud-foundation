# Get-VCFUpgrade

### Synopsis
Get the Upgrade

### Syntax
```
Get-VCFUpgrade
```

### Description
The Get-VCFUpgrade cmdlet retrives a list of upgradable resources in SDDC Manager

### Examples
#### Example 1
```
Get-VCFUpgrade
```
This example shows how to retrieve the list of upgradable resources in the system
